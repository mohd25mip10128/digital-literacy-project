## 🛡️ Cyber Safety Tips  
- Use strong passwords  
- Avoid clicking unknown links  
- Do not share OTPs or personal data  
- Keep software updated  
- Use secure payment methods  

Report cybercrime:  
- 🌐 https://cybercrime.gov.in  
- 📞 Helpline: 1930
